# 📰 뉴스 스크래퍼 Pro v31.1

네이버 뉴스 API를 활용한 데스크톱 뉴스 수집 및 관리 프로그램입니다.

![Python](https://img.shields.io/badge/Python-3.8+-blue)
![PyQt6](https://img.shields.io/badge/PyQt6-6.0+-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

---

## ✨ 주요 기능

### 📡 뉴스 수집
- 네이버 검색 API를 통한 실시간 뉴스 검색
- 키워드별 탭 분리로 효율적 관리
- 자동 새로고침 (10분~6시간 간격)
- 중복 기사 자동 감지

### 🔔 알림 시스템
- **데스크톱 알림**: 새 뉴스 도착 시 시스템 트레이 알림
- **키워드 알림**: 특정 키워드 포함 기사 별도 알림
- **탭 배지**: 미읽음 기사 수 실시간 표시 `(5)`

### 📚 기사 관리
- 읽음/안읽음 상태 관리
- ⭐ 북마크 기능
- 📝 메모 작성
- 필터링 및 정렬
- **"더 보기" 기능**: 대용량 데이터 효율적 로딩

### 💾 데이터 관리
- SQLite 데이터베이스 저장
- CSV 내보내기
- JSON 설정 백업/복원
- **자동 DB 인덱스 최적화**

### 🎨 UI/UX
- 라이트/다크 테마 지원
- 현대적인 카드 UI (그라디언트, 호버 효과)
- 유형별 토스트 메시지 (성공/경고/오류)
- 상태바 실시간 카운트다운 표시
- 검색 히스토리 자동 저장
- **첫 실행 가이드**: API 키 설정 안내
- **툴팁 단축키 힌트**

### 🛡️ 안정성
- **종료 원인 추적**: 예기치 않은 종료 시 원인 로깅
- **종료 확인 다이얼로그**: 실수로 종료 방지
- **Windows 세션 종료 감지**
- **SIGTERM/SIGINT 핸들링**

## 🚀 설치 방법

### 요구사항
- Python 3.8 이상
- 네이버 개발자 API 키

### 설치
```bash
pip install PyQt6 requests
```

### 실행
```bash
python "251218 네이버 뉴스 자동검색 vfinal-디버깅.py"
```

### 빌드 (실행 파일 생성)
```bash
pip install pyinstaller
pyinstaller news_scraper.spec
```

---

## ⚙️ 네이버 API 키 발급

1. [네이버 개발자센터](https://developers.naver.com/) 접속
2. 애플리케이션 등록 → "검색" API 선택
3. Client ID / Client Secret 발급
4. 프로그램 설정에서 입력

> 💡 **첫 실행 시** API 키가 없으면 자동으로 설정 안내 다이얼로그가 표시됩니다.

## ⌨️ 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+R` / `F5` | 새로고침 |
| `Ctrl+T` | 새 탭 추가 |
| `Ctrl+W` | 현재 탭 닫기 |
| `Ctrl+F` | 필터 검색 |
| `Ctrl+S` | CSV 내보내기 |
| `Ctrl+,` | 설정 열기 |
| `Alt+1~9` | 탭 전환 |
| `F1` | 도움말 |

> 💡 **툴팁 힌트**: 각 버튼에 마우스를 올리면 단축키를 확인할 수 있습니다.

## 🚀 성능 최적화

### HTML 렌더링 최적화
| 설정 | 값 |
|------|----|
| 초기 렌더링 | 50개 |
| 추가 로딩 | 30개씩 |
| 최대 렌더링 | 500개 |

### DB 인덱스 최적화
다음 복합 인덱스가 자동 생성됩니다:
- `keyword + is_read`
- `keyword + pubDate_ts`
- `keyword + is_duplicate`
- `is_bookmarked + pubDate_ts`
