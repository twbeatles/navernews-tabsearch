# 📰 News Scraper Pro (v32.1)

**뉴스 스크래퍼 Pro**는 네이버 뉴스 API를 활용하여 원하는 키워드의 뉴스를 실시간으로 모니터링하고, 필터링 및 관리할 수 있는 강력한 데스크톱 애플리케이션입니다.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![PyQt6](https://img.shields.io/badge/GUI-PyQt6-green)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

---

## ✨ 주요 기능 (Key Features)

### 1. 🎨 모던 프리미엄 UI/UX
- **Apple-Style 디자인**: 깔끔하고 세련된 인터페이스로 가독성을 극대화했습니다.
- **다크 모드 지원**: 눈이 편안한 다크 모드와 기본 라이트 모드를 완벽하게 지원합니다.
- **반응형 카드 레이아웃**: 뉴스 기사가 카드 형태로 표시되어 직관적입니다.

### 2. 🚀 강력한 뉴스 검색 및 관리
- **실시간 모니터링**: 설정된 간격으로 자동으로 뉴스를 새로고침합니다.
- **탭 기반 검색**: 여러 키워드를 탭으로 관리하며 동시에 모니터링할 수 있습니다.
- **고급 필터링**:
  - **제외 키워드**: `-광고` 처럼 입력하여 원하지 않는 기사를 숨길 수 있습니다.
  - **실시간 검색**: 결과 내에서 즉시 제목/내용을 검색할 수 있습니다.
  - **중복 제거**: 유사한 기사를 자동으로 감지하여 그룹화합니다.

### 3. 📂 데이터 관리 및 편의성
- **북마크 기능**: 중요한 기사는 별도로 저장하여 언제든 다시 볼 수 있습니다.
- **읽음 상태 관리**: 읽은 기사는 흐릿하게 표시되어 안 읽은 기사에 집중할 수 있습니다.
- **엑셀 내보내기**: 검색된 결과를 엑셀 파일(.csv)로 저장할 수 있습니다.
- **자동 백업**: 설정과 데이터를 매일 자동으로 백업하여 안전하게 보관합니다.

### 4. 🔗 미리보기 및 공유
- **마우스 오버 미리보기**: 링크를 클릭하지 않고도 내용을 미리 볼 수 있습니다.
- **외부 브라우저 열기**: 원문 기사를 기본 브라우저에서 바로 열 수 있습니다.
- **공유 기능**: 기사 링크를 클립보드에 쉽게 복사할 수 있습니다.

## 🛠️ 설치 및 실행 (Installation)

### 실행 파일(EXE)로 실행하기
1. `dist/news_scraper_pro.exe` 파일을 다운로드합니다.
2. 프로그램을 실행하고 네이버 API 키를 설정합니다.

### 소스 코드로 실행하기
1. 파이썬 3.8 이상을 설치합니다.
2. 필요한 라이브러리를 설치합니다:
   ```bash
   pip install PyQt6 requests
   ```
3. 프로그램을 실행합니다:
   ```bash
   python news_scraper_pro.py
   ```

---

## 🔑 초기 설정 (API Key)

이 프로그램은 **네이버 검색 API**를 사용합니다.
1. [네이버 개발자 센터](https://developers.naver.com/)에 접속하여 로그인합니다.
2. **Application 등록**을 클릭합니다.
   - **사용 API**: `검색`
   - **비로그인 오픈 API 서비스 환경**: `WEB 설정` (URL은 `http://localhost` 입력)
3. 발급받은 **Client ID**와 **Client Secret**을 프로그램 설정 창에 입력합니다.

---

## 🏗️ 빌드 방법 (Building)

직접 실행 파일(.exe)을 만들고 싶다면 `PyInstaller`를 사용하세요.

1. PyInstaller 설치:
   ```bash
   pip install pyinstaller
   ```
2. 빌드 실행:
   ```bash
   pyinstaller news_scraper.spec
   ```
3. `dist` 폴더에 생성된 `news_scraper_pro.exe` 확인.
