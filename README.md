# 뉴스 스크래퍼 Pro v32.1.1

네이버 뉴스 API를 활용한 탭 기반 뉴스 스크래퍼입니다.

> [!NOTE]
> **Hotfix v32.1.1**: 프로그램 초기화 중 오류 발생 시 비정상 종료되는 문제를 수정했습니다. (MainApp 안정성 개선)

## 주요 기능

### 핵심 기능
- **탭 기반 키워드 검색**: 여러 키워드를 탭으로 관리
- **제외 키워드**: `-`로 시작하는 키워드는 검색 결과에서 제외
- **자동 새로고침**: 10분~6시간 간격 설정 가능
- **북마크 & 메모**: 중요 기사 저장 및 메모 작성
- **라이트/다크 테마**: 시스템과 독립적인 테마 선택
- **중복 감지**: 유사 제목 기사 자동 탐지

### 알림 기능
- **데스크톱 알림**: 새 기사 도착 시 알림
- **알림 키워드**: 특정 키워드 포함 시 강조 알림
- **자동 새로고침 알림**: 새로고침 완료 시 알림 (옵션)

### 시스템 통합
- **시스템 트레이**: X 버튼 클릭 시 트레이로 최소화
- **윈도우 자동 시작**: 부팅 시 자동 실행 옵션
- **창 상태 기억**: 창 위치/크기 저장 및 복원

### 데이터 관리
- **CSV 내보내기**: 검색 결과를 CSV로 저장
- **키워드 그룹**: 자주 사용하는 키워드 그룹 관리
- **자동 백업**: 설정 자동 백업 및 복원

### UX 개선
- **휠 스크롤 방지**: 설정창에서 휠로 값 변경 방지
- **성능 최적화**: 정규식 프리컴파일, 패턴 캐시

## 설치 및 실행

### 요구사항
- Python 3.9+
- PyQt6
- requests

### 설치
```bash
pip install PyQt6 requests
```

### 실행
```bash
python news_scraper_pro.py
```

## API 키 설정

1. [네이버 개발자센터](https://developers.naver.com/apps/#/register)에서 애플리케이션 등록
2. 검색 API (뉴스 검색) 선택
3. 프로그램 실행 후 설정에서 Client ID/Secret 입력

## 단축키

| 단축키 | 기능 |
|--------|------|
| `Ctrl+R` / `F5` | 새로고침 |
| `Ctrl+T` | 새 탭 추가 |
| `Ctrl+W` | 현재 탭 닫기 |
| `Ctrl+F` | 필터 검색창 포커스 |
| `Ctrl+S` | CSV 내보내기 |
| `Ctrl+,` | 설정 |
| `Alt+1~9` | 탭 전환 |
