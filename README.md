# Naver News Tab Search v1.5.0

네이버 뉴스 검색 및 모니터링을 위한 고성능 데스크톱 애플리케이션입니다. 탭 기반 인터페이스로 여러 키워드를 동시에 관리하고, 실시간 알림 및 데이터 분석 기능을 제공합니다.

## ✨ 주요 기능

*   **탭 기반 멀티 키워드 검색**: 여러 키워드를 각각의 탭으로 관리하며 독립적으로 검색 및 모니터링
*   **실시간 자동 새로고침**: 설정한 간격으로 뉴스 자동 업데이트 및 데스크톱/토스트 알림
*   **시각적 데이터 분석**: 언론사별 보도량, 시간대별 트렌드 등을 시각적으로 분석 (Matplotlib 통합)
*   **고급 필터링**: 읽은 기사 제외, 북마크, 키워드 하이라이팅, 중복 기사 자동 감지
*   **데이터 관리**: CSV 내보내기, 백업/복원, 키워드 그룹 관리
*   **시스템 최적화**: 다크 모드 지원, HiDPI 대응, 메모리 최적화, 스레드 풀 사용

## 🛠 v1.5.0 업데이트 내역

*   **UI 개선**: 툴바 버튼 간격 및 스타일 조정으로 버튼 겹침 현상 해결
*   **안정성 강화**: 초기화 과정의 안전장치 추가로 설정 오류 시에도 프로그램이 크래시되지 않도록 수정
*   **최적화**: 불필요한 디버그 코드 제거 및 데이터베이스 연결 관리 개선

## 📦 설치 및 실행

### 필수 요구 사항
*   Python 3.9 이상
*   Chrome 브라우저 (내장 브라우저 엔진용)

### 라이브러리 설치
```bash
pip install -r requirements.txt
```

### 실행 방법
```bash
python news_scraper_pro.py
```

## 🏗 빌드 (exe 파일 생성)

PyInstaller를 사용하여 단일 실행 파일로 빌드할 수 있습니다. 이미 최적화된 `.spec` 파일이 포함되어 있습니다.

```bash
pip install pyinstaller
pyinstaller news_scraper.spec
```

빌드가 완료되면 `dist/NaverNewsTabSearch` 폴더에 실행 파일이 생성됩니다.
